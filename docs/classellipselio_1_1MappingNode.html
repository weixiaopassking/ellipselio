Detailed Description

Main ROS 2 node implementing LiDAR-inertial odometry with tensor voting.

Orchestrates multi-sensor fusion, implements tensor voting algorithm for ellipsoid-based map representation, performs EKF state estimation, and publishes odometry, map, and analytics information.
