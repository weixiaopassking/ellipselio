Detailed Description

EllipseLio unified point type with geometry, color, intensity, and metadata.

Combines position (x,y,z), RGBA color, intensity, timestamp, bin index, scan index, and primitive type for the ellipsoid representation. Aligned to 16 bytes for SIMD optimization.
