Detailed Description

LiDAR point cloud processing with vendor-specific support.

Processes point clouds from multiple LiDAR vendors (Livox, Velodyne, Ouster, Hesai, Gazebo), handles temporal synchronization, range binning with octree indexing, and point undistortion.
